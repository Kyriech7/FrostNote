# 设计规范

## 设计方向

界面应简洁、轻量、安静，像一张随时可用的桌面便签纸。视觉重点是白色、天蓝色和轻微毛玻璃质感，不做厚重、复杂或装饰过多的界面。

应用品牌名为 FrostNote。图标使用蓝色圆角底、白色便签纸、雪花和冰晶元素，保持冰霜、清爽、毛玻璃的视觉联想；应用内品牌块和 Windows 图标应使用同一套图形资产。

## 色彩

- 主背景：接近白色。
- 主强调色：天蓝色。
- 危险或逾期提示：清晰红色。
- 已完成状态：降低对比度，并使用删除线。
- 边框和分隔线：低对比度浅灰或浅蓝。

## 毛玻璃效果

- 使用半透明浅色面板。
- 使用轻微背景模糊。
- 阴影要克制，避免厚重卡片感。
- 毛玻璃用于主要容器和浮层，不要滥用在每个小元素上。

## 布局

- 默认聚焦今天。
- 使用日期侧栏或顶部日期切换区域。
- 中央区域展示当天记录。
- 快速输入区域始终容易找到。
- 搜索和筛选保持轻量，不抢占记录主体。

## 交互

- 新增记录要少步骤。
- to do 右侧对勾按钮应清楚、易点击。
- 完成事项保留在列表中，并显示删除线。
- 逾期滚动事项用红色文字提示，不弹窗打扰。
- 删除操作需要避免误触，可使用二次确认或撤销机制。

## 可访问性

- 文字与背景保持足够对比度。
- 红色逾期状态不要只依赖颜色，后续可增加小型状态标记。
- 按钮需要有明确 hover、focus 和 disabled 状态。

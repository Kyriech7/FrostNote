## 问题/风险

- 当前目录不是 Git 仓库，本轮未创建提交记录。
- PowerShell 执行策略会阻止 `npm.ps1`，项目命令优先使用 `npm.cmd`。
- Rust 路径已写入用户 Path，但已有终端可能需要重新打开后才能直接识别 `cargo`。
- VS Build Tools 虽指定安装到 `Z:`，微软安装器仍可能在系统盘写入必要系统组件、注册表或缓存。
- 无装饰窗口的按钮坐标会受窗口缩放与阴影影响，后续如调整顶部控件尺寸，需要重新做真实窗口点击验证。
